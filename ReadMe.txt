throttle is a C++ class that implements "throttling" mechanism.

throttle - это С++ класс, который позволяет реализовать "throttling" (ограничение скорости какого-нибудь процесса): не более count вызовов некоторой функции в течении заданного периода времени interval.

TestThrottle.sln - тестовое приложение (для Visual Studio 2010), в котором можно посмотреть пример использования класса throttle.
